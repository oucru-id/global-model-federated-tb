# TB FHIR Phylogeny Analysis Pipeline (Global) - In Development

This pipeline processes TB SNP Distance Matrix and Newick Tree to generate global distance matrices, phylogenetic trees, and transmission network visualizations. Please refer to our full documentation: https://tb-pipeline-docs.readthedocs.io/en/latest/index.html

### Input

- Distance Matrix `data/distance_matrix/`
- Newick Tree `data/nwk_tree/`
